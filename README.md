# 🖼️ Python GIF Creator

A simple Python project that creates an animated GIF from multiple images using the Pillow library.

## ✨ Features

- Create GIFs from JPG, JPEG, and PNG images
- Automatically reads images from the `images` folder
- Preserves the aspect ratio of images
- Centers images with padding
- Saves the GIF in the `output` folder

## 📁 Project Structure

```
python-gif-creator/
│
├── images/
│   ├── 1.jpg
│   ├── 2.jpg
│   ├── 3.jpg
│   ├── 4.jpg
│   ├── 5.jpg
│   └── 6.jpg
│
├── output/
│   └── animation.gif
│
├── main.py
├── requirements.txt
├── .gitignore
└── README.md
```

## 🛠️ Technologies Used

- Python 3
- Pillow (PIL)

## 🚀 Installation

```bash
pip install pillow
```

## ▶️ Run the Project

```bash
python main.py
```

The generated GIF will be saved in the `output` folder.

## 📷 Preview

After running the project:

- Input: Images placed in the `images` folder
- Output: `output/animation.gif`
